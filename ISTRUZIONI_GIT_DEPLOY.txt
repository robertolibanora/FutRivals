# ISTRUZIONI GIT + DEPLOY FUTRIVALS

# Per ogni modifica ai dati, immagini o codice:
1. Aggiungi tutti i file modificati:
   git add .
2. (Consigliato) Aggiungi esplicitamente i file Excel e le immagini loghi/sponsor:
   git add FUTRIVALSDATA.xlsx DATITORNEO.xlsx
   git add static/squadre/*.png
   git add static/*.png
   git add static/*.pdf
3. Crea un commit con un messaggio descrittivo:
   git commit -m "Messaggio descrittivo delle modifiche"
4. Esegui sempre il push per inviare le modifiche su GitHub e attivare il deploy su Render:
   git push

# IMPORTANTE:
- Il deploy su Render parte solo dopo il push su GitHub.
- Se aggiorni Excel, immagini o PDF, ricordati di aggiungerli e pusharli SEMPRE.
- Puoi forzare il deploy dal pannello Render con "Manual Deploy" se necessario.
